PPP Compression Commands

Point-to-point software compression on serial interfaces can be configured after PPP encapsulation is enabled. Because this option invokes a software compression process, it can affect system performance. If the traffic already consists of compressed files, such as .zip, .tar, or .mpeg, do not use this option. The figure shows the command syntax for the compress command.

Figure shows two routers connecting to a WAN cloud using P.P.P. The figure shows the output from the configuration file. The line of output that reads compress predictor is highlighted. This line configures the predictor compression algorithm on the serial interface. The other compression option is the stacker algorithm and would be configured with the command compress stac.
