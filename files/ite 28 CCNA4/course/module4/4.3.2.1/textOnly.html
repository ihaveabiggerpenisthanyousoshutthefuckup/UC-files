Figure 1 shows the topology that will be used for configuring IPv6 ACLs. The topology is similar to the previous IPv4 topology except for the IPv6 addressing scheme. There are three 2001:DB8:CAFE::/64 subnets:

  • 2001:DB8:CAFE:10::/64

  • 2001:DB8:CAFE:11::/64

  • 2001:DB8:CAFE:30::/64

Two serial networks connect the three routers:

  • 2001:DB8:FEED:1::/64

  • 2001:DB8:FEED:2::/64

Figures 2, 3, and 4 show the IPv6 address configuration for each router. The show ipv6 interface brief command is used to verify the address and the state of the interface.

Note: The no shutdown command and the clock rate command are not shown.
