Verifying an IP SLA Configuration

Use the show ip sla configuration operation-number command to display configuration values including all defaults for IP SLA operations or for a specific operation.

In Figure 1, the show ip sla configuration command displays the IP SLA ICMP Echo configuration.

Use the show ip sla statistics [operation-number] command to display the IP SLA operation monitoring statistics, as shown in Figure 2.
