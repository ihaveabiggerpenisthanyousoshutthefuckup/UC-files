Verify OSPFv3 Interfaces

The quickest way to verify OSPFv3 interface settings is to use the show ipv6 ospf interface command. This command provides a detailed list for every OSPFv3-enabled interface.

To retrieve and view a summary of OSPFv3-enabled interfaces on R1, use the show ipv6 ospf interface brief command, as shown in Figure 1.

Use the Syntax Checker in Figure 2 to view a summary of OSPFv3-enabled interfaces on R2 using the show ipv6 ospf interface brief command. Notice that specifying the interface name as done in the show ipv6 ospf interface serial 0/0/1 command provides detailed OSPFv3 information.

Continue using the Syntax Checker in Figure 2 to get a summary of OSPFv3-enabled interfaces on R3 using the show ipv6 ospf interface brief command. Retrieve and view additional information for the Serial 0/0/0 interface using the show ipv6 ospf interface serial 0/0/0 command.

Figure one shows the output of the following command run on router R-1, show i.p.v.6. O.S.P.F. interface brief. The result is a detailed list for every O.S.P.F. enabled interface Figure two is an interactive media element that allows the learner to verify the O.S.P.F.v.3. interfaces on router R-2.
