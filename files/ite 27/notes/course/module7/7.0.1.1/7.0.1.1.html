Chapter 7: EIGRP Tuning and Troubleshooting

EIGRP is a versatile routing protocol that can be fine-tuned in many ways. Two of the most important tuning capabilities are the ability to summarize routes and the ability to implement load balancing. Other tuning capabilities include being able to propagate a default route, fine-tune timers, and implement authentication between EIGRP neighbors to increase security.

This chapter discusses these additional tuning features and the configuration mode commands to implement these features for both IPv4 and IPv6.
