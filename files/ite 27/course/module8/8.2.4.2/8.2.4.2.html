Verify OSPF Protocol Settings

As shown in Figure 1, the show ip protocols command is a quick way to verify vital OSPF configuration information. This includes the OSPFv2 process ID, the router ID, networks the router is advertising, the neighbors the router is receiving updates from, and the default administrative distance, which is 110 for OSPF.

Use the Syntax Checker in Figure 2 to verify the OSPFv2 protocol settings of R2 and R3 using the show ip protocols command.

Figure one shows the output from the following command run on router R-1, show i.p. protocols. The output shows the O.S.P.F. process I.D., the router I.D., networks the router is advertising, the neighbors the router is receiving updates from, and the default administrative distance. Figure two is an interactive media element that allows the learner to verify O.S.P.F. protocol settings.
