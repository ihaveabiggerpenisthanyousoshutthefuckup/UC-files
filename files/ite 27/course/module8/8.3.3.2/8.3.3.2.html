Verify OSPFv3 Protocol Settings

As shown in Figure 1, the show ipv6 protocols command is a quick way to verify vital OSPFv3 configuration information, including the OSPFv3 process ID, the router ID, and the interfaces enabled for OSPFv3.

Use the Syntax Checker in Figure 2 to verify the OSPFv3 protocol settings of R2 and R3 using the show ipv6 protocols command.

Use the show ipv6 ospf command to also examine the OSPFv3 process ID and router ID. This command displays the OSPFv3 area information and the last time the SPF algorithm was calculated.

Figure one shows the output from the following command run on router R-1, show i.p.v.6. protocols. The output lists the following vital O.S.P.F.v.3. configuration information: the O.S.P.F. process I.D., the router I.D., and the interfaces enabled for O.S.P.F. version 3.Figure two is an interactive media element that allows the learner to verify O.S.P.F.v.3. protocol settings for router R-2.
