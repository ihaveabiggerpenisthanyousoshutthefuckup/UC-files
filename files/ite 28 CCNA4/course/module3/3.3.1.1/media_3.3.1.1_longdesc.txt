The figure shows four different VPN connections through an Internet cloud going back to the main corporate building, the main site. The main site has a Cisco ASA firewall used to connect to the Corporate cloud or network. The first type of VPN connection to the main site is a business partner with a Cisco router. The second type of VPN is a regional office with a Cisco ASA firewall. The third type of connection is a SOHO with a Cisco router. The last type of VPN connection is a mobile worker laptop with Cisco AnyConnect installed.
