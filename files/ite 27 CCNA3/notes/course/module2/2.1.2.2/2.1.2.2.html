Step 1 – Configure the VTP Server

Confirm that all switches are configured with default settings to avoid any issues with configuration revision numbers. Configure S1 as the VTP server using the vtp mode server global configuration command, as shown in Figure 1

Issue the show vtp status command to confirm that S1 is a VTP server, as shown in Figure 2.

Notice how the configuration revision number is still set to 0 and the number of existing VLANs is 5. This is because no VLANs have yet been configured and the switch does not belong to a VTP domain. The 5 VLANs are the default VLAN 1 and VLANs 1002-1005.
